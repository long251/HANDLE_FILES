*Domain là gì? Bạn đã thật sự hiểu về domain: ý nghĩa, cấu trúc, vai trò
- ý nghĩa: địa chỉ web -> miền -> tên miền
- cấu trúc: - tên miền. hậu tố (.com / .vn...): google.com
	    - miền phụ.tên miền.hậu tố: mail.google.com (sub domain + tên miền + hậu tố)
	    - hậu tố: 1 từ: .com/.vn/.org ; 2 từ .com.vn
	    - hậu tố cuối cùng là TLD; thứ 2 là Second level domain
	    - cấu trúc hình cây: từ gốc: root > TLD > SLD > DN > SD
- vai trò: dễ nhớ
-------------------------------
*Mô hình client server:
- kết nối máy tính với nhau
- client-máy khách: PC (máy tính cá nhân), smart phone __ request
- Server-máy chủ: chịu tải Vd: live server trong VScode __ respone
- gửi yêu cầu TCP/IP/ HTTP; kết nối rất nhanh; máy chủ luôn sẵn sàng kết nối
